Хук useParams в React Router: Получение параметров из URL
useParams — это встроенный хук библиотеки React Router, который используется для чтения динамических параметров из текущего URL-адреса. Это критически важный инструмент для создания динамических страниц, таких как карточки товаров, профили пользователей или статьи блога.
Как это работает
Когда вы определяете маршрут (Route), вы можете указать в пути динамическую часть, используя двоеточие (например, :id или :slug). Хук useParams возвращает объект, в котором ключи — это названия параметров из настроек роута, а значения — то, что фактически находится в адресной строке браузера.
Пример лаконичного кода
Представьте, что у вас есть роут, настроенный как /product/:productId.

import { useParams } from 'react-router-dom';

function ProductPage() {
  // Извлекаем параметр productId из URL
  const { productId } = useParams();

  return (
    <div>
      <h1>Просмотр товара</h1>
      <p>ID выбранного товара: {productId}</p>
    </div>
  );
}

Основные сценарии использования

    Загрузка данных: Получив ID из URL, вы можете отправить запрос к API (например, через useEffect) для получения данных именно об этом объекте.
    Навигация: Использование параметров позволяет одному и тому же компоненту отображать разный контент в зависимости от адреса.
    Фильтрация: Параметры могут использоваться для определения категорий или типов данных, которые нужно отобразить на странице.

Важные особенности

    Реактивность: Если параметры в URL изменятся (например, пользователь перейдет с /product/1 на /product/2), компонент, использующий useParams, автоматически перерисуется с новым значением.
    Типизация: При использовании TypeScript (который изучается в продвинутых модулях) можно типизировать возвращаемый объект параметров для более безопасной разработки.
    Место вызова: Как и любой хук, useParams должен вызываться только на верхнем уровне функционального компонента, который находится внутри провайдера маршрутов (например, createBrowserRouter).
