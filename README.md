## О проекте

> **От простых компонентов к мощным приложениям: React, TypeScript и современная разработка**

Этот репозиторий предназначен для комплексного изучения React с нуля. Основной упор сделан на понимание компонентов, хуков, управления состоянием и использовании TypeScript для безопасности типов. Каждый проект полностью независим с собственной конфигурацией Vite и TypeScript.

## Быстрый старт

### Запуск упражнений

```bash
# 1. Клонирование репозитория
git clone <repository-url>
cd -React_Education-

# 2. Переход в упражнение
cd cd Practical_Tasks/00_nameProject

# 3. Установка зависимостей и запуск
npm install && npm run dev
```

### Запуск проектов

```bash
cd Practical_Tasks/00_nameProject
npm install && npm run dev
```

## Команды

```bash
# В папке каждого проекта/упражнения
npm run dev        # Запуск dev сервера (http://localhost:3000)
npm run build      # Production сборка
npm run type-check # Проверка типов TypeScript
npm run lint       # Проверка ESLint
```
